GIT

GIT is a Distributed Version Control System Some commands are

create a new repository on the command line

echo “# website” >> README.md

git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/Username/website.git
git push -u origin master

or push an existing repository from the command line

git remote add origin https://github.com/Username/website.git
git push -u origin master

View remote and remove

git remote -v
git remote rm destination
git remote -v

Vagrant

Vagrant from HashiCorp is a good tool for spinning up environments and test.

Can be used with Virtual Box,VMware and there are other providers also like Docker and Hyper-V.

Some commands

## vagrant init
## vagrant up
## vagrant ssh  hostname
## vagrant halt

Vagrantfile sample

Vagrant.configure("2") do |config|

 config.vm.define "web" do |web|
 
 web.vm.box = "bento/centos-7.2"
 
 web.vm.hostname = 'centos1'
 
 web.vm.box_url = "bento/centos-7.2"
 
 web.vm.network "private_network", ip: "192.168.33.10"
 
 web.vm.network :forwarded_port, guest: 22, host: 10123, id: "ssh"
 

end
 
config.vm.define "app" do |app|
 
app.vm.box = "bento/centos-7.2"

app.vm.hostname = 'centos2'

app.vm.box_url = "bento/centos-7.2"

app.vm.network "private_network", ip: "192.168.33.11"

app.vm.network :forwarded_port, guest: 22, host: 10124, id: "ssh"

      
end
