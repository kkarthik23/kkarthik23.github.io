Hugo static generator

2017/08/26

Hugo is a static generator Some settings needs to be configured to generate output and push to github

In cofig.toml set baseURL = “https://username.github.io/"

run hugo

c:\hugo\quickstart\hugo

files are generated under public folder type c:\hugo\quickstart cd public and then run gitpush.bat which contains below

git init

git add .

git commit -m “first commit”

git remote add origin https://github.com/username/username.github.io

git push -u origin master
