Nagios

10/07/2017

Nagios is a monitoring system

Install nagios on Centos 7

Server Configuration:

Install EPEL:

# yum install epel-release.noarch -y


# yum --enablerepo=epel -y install nagios


# yum --enablerepo=epel -y install nagios-plugins-disk nagios-plugins-http nagios-plugins-load nagios-plugins-ping nagios-plugins-procs nagios-plugins-ssh nagios-plugins-swap nagios-plugins-users



# Add nagios admin user

# htpasswd /etc/nagios/passwd nagiosadmin

New password:

Re-type new password:

Adding password for user nagiosadmin

Enable the nagios service:


# systemctl enable nagios

# systemctl start nagios

# systemctl enable httpd

# systemctl restart httpd

Client Configuration:

Install EPEL:


# yum install epel-release.noarch -y

# yum --enablerepo=epel -y install nrpe nagios-plugins-{ping,disk,users,procs,load,swap,ssh}

Edit /etc/nagios/nrpe.cfg file,


# vim /etc/nagios/nrpe.cfg

Add the ip address of your Nagios Server:

Find the following line

allowed_hosts=127.0.0.1

Add the ip address of the Nagios server

allowed_hosts=127.0.0.1,192.168.1.150

Enable and start nrpe service on the client:


# systemctl start nrpe

# chkconfig nrpe on
