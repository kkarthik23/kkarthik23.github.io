Notepad++ extension

2017/11/08

Notepad++ comes with the default theme and it is white background, to change it Download Material Theme from https://github.com/naderi/material-theme-for-npp

Save it as xml in #%APPDATA%\Notepad++, under themes folder.

If it doesn’t exist then create the themes folder Goto Settings,Style Configurator and select the new theme
