Django Basics

Creating a new Django Project

To create the project run,

python django-admin.py startproject

the name of the project you wish to create.

Creating a new Django App

  1. To create a new app, run 

    $ python manage.py startapp <appname>

    name of the app you wish to create.

  2. Tell your Django project about the new app by adding it to the tuple in your INSTALLED_APPS project’s file settings.py

  3. In your project file, add a mapping to the app.urls.py

  4. In your app’s directory, create a file to direct incoming URL strings to views. urls.py

  5. In your app’s py, create the required views ensuring that they return a view HttpResponse object.
